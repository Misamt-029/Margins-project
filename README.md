# Margins-project

This repository contains a static copy of the `MARGINS` app from https://marginsmisa.lovable.app.

## Preview

Open `index.html` in a browser to view the app locally.

## Contents

- `index.html` — main entry point
- `archive.html`, `create.html`, `desk.html`, `notes.html`, `threads.html` — additional site pages
- `assets/` — compiled CSS and JavaScript bundles
- `__l5e/events.js`, `~flock.js` — site runtime scripts

## Notes

This repo stores the generated/static version of the site as fetched from the live app.

## Local Preview

1. Install dependencies:

   ```bash
   npm install
   ```

2. Start the static preview server:

   ```bash
   npm start
   ```

Then open the local URL shown in the terminal.
